# Tempo — AI 日程排程助手

> 产品定位：AI 帮你自动排好一天的日程——你只管输入任务，剩下的 AI 来安排。

## 基本信息

- **状态**: Sprint 1 筹备中
- **GitHub**: 待建
- **技术栈**: Flutter(iOS) / FastAPI / OpenAI / Google Calendar API / Stripe
- **目标市场**: Clockwise 关停后的 ToC 替代品，欧美 + 中文空白市场

## 产品方向（已确认，2026-03-31）

### 核心差异化
1. **iOS 原生**（Motion 移动端 G2 评分 2.7/5 是竞品共同弱点）
2. **免费版可用**（竞品均无免费 AI 排程）
3. **中文市场空白**（国内无 AI 排程产品）
4. **5 分钟上手**（vs Motion 2-4 周上手）

### 定价方案
| 层级 | 价格 | 功能 |
|------|------|------|
| Free | $0 | 基础日历 + 每天 3 个 AI 排程任务 |
| Pro | $7.99/mo | 无限 AI 排程 + 智能重排 + 时间块分析 + 周报 |
| Team | $12.99/user/mo | 共享日历 + 会议协调 + 团队精力分析 |

## Sprint 规划

### Sprint 1（Week 1-2）: 核心排程引擎
- [ ] 自然语言输入任务
- [ ] AI 自动排入日历（优先级 + 截止日期 + 时间偏好）
- [ ] 日/周/月视图
- [ ] 基础冲突检测

- [ ] 通知提醒

### Sprint 3（Week 5-6）: 变现与增长

## 市场机会

- **Clockwise 2026-03-27 已关停**，大量用户正在找替代品
