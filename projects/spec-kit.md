# spec-kit

GitHub spec-kit 规格驱动开发工作流

## 基本信息

- **状态**: 新建
- **创建日期**: 2026-04-22
- **技术栈**: Python, specify-cli

## 项目描述

使用 GitHub spec-kit 进行规格驱动开发（Spec-Driven Development）的工作流。

## 工具安装

```bash
uv tool install specify-cli --from git+https://github.com/github/spec-kit.git
```

## 工作流程

1. `/speckit-constitution` - 建立项目原则
2. `/speckit-specify` - 创建功能规格
3. `/speckit-clarify` - 澄清需求
4. `/speckit-plan` - 创建技术计划
5. `/speckit-tasks` - 生成任务
6. `/speckit-implement` - 执行实现
